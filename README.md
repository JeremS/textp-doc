

# Textp-doc
This library provides some helpers when making a project's documentation.

## Installation
Deps coords:
```clojure
#:fr.jeremyschoffen.textp{doc-alpha #:mvn{:version "0"}}
```
Lein coords:
```clojure
[fr.jeremyschoffen.textp/doc-alpha "0"]
```
